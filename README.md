# 🌱 Paradise Nursery Shopping Application

A modern React-based shopping cart application for an online plant nursery. This project allows users to browse plants, add them to a cart, and manage their selections efficiently.

---

## 🚀 Features

* 🌿 Display a list of available plants
* 🛒 Add plants to the shopping cart
* ➕ Increase / ➖ decrease item quantity
* ❌ Remove items from cart
* 💰 Calculate total price dynamically
* ⚡ Fast performance using React + Vite
* 🧠 State management using Redux Toolkit

---

## 🛠️ Tech Stack

* **React (Vite)**
* **Redux Toolkit**
* **React-Redux**
* **JavaScript (ES6+)**
* **HTML & CSS**

## ▶️ Getting Started

### 1. Clone the repository

```
git clone https://github.com/your-username/paradise-nursery.git
cd paradise-nursery
```

### 2. Install dependencies

```
npm install
```

### 3. Run the app

```
npm run dev
```

### 4. Open in browser

```
http://localhost:5173
```
